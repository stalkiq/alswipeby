# **App Name**: Spreadsheet Saver

## Core Features:

- Data Input: Allow users to input data into a table with columns for Business Name, Street, City, ZIP, Phone, Category, Facebook URL, Facebook Last Post, Instagram URL, Instagram Present, Website, and Online on.
- Data Persistence: Store user-entered data in Firestore to allow for retrieval across sessions.
- Data Saving: Provide a button that saves current table data.

## Style Guidelines:

- Primary color: A gentle blue (#64B5F6) to evoke trust and reliability.
- Background color: A light, desaturated blue (#E3F2FD) to maintain a clean and professional look.
- Accent color: A muted green (#81C784) to provide visual contrast and highlight actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif, will be used throughout for its clean, modern appearance.
- Use a clean and tabular layout to display the data, mimicking the structure of a spreadsheet. Ensure responsiveness across different devices.
